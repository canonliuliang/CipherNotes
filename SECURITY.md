# Security Policy

密笺 CipherNotes 是本地优先的加密记事本与文件保险柜。安全问题请谨慎报告，避免在公开 issue 中粘贴个人数据、保险库文件、恢复码、密码、日志中的路径信息或真实隐私内容。

## Supported Versions

当前公开版本会优先处理安全问题。旧版本建议升级到最新 Release。

## Reporting a Vulnerability

如果你发现以下问题，请通过 GitHub issue 先提交一个不含敏感数据的简短报告：

- 保险库或保险柜文件可能泄露明文内容
- 密码、恢复码或本地密钥处理存在绕过风险
- 自动锁定、用户隔离、管理员权限边界存在明显漏洞
- 导入、导出、备份或还原可能访问了未授权文件

报告时请尽量包含：

- macOS 版本
- 密笺版本
- 复现步骤
- 预期行为和实际行为
- 是否涉及真实隐私数据

如果漏洞涉及真实敏感数据，请先只描述影响范围，不要上传原始文件。维护者会再确认更安全的沟通方式。

## Security Boundaries

- 管理员密码不是用户内容的万能解锁钥匙。
- 忘记用户密码且没有恢复码时，该用户数据无法恢复。
- 密笺不提供云端恢复；多人共用同一台 Mac 时，建议为每个系统用户单独使用密笺。
- 应用不提供云同步；备份文件的保管责任在用户自己。
