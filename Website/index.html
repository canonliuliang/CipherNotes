为什么选择 CipherNotes？

它不是另一个把数据搬进云端的笔记工具，而是给本机私密资料准备的安全工作台。

隐私边界更清楚

没有远程账号、广告分析、会员系统和同步服务器。数据留在本机，备份也由你自己决定放在哪里。

笔记和文件一起保护

不只加密文字。照片、PDF、压缩包、扫描件和大文件都能进入保险柜，形成一套完整的本地资料库。

面向真实大文件

保险柜使用分片加密和后台导入，10GB 级图片或素材包也不会把主界面拖住。

没有万能钥匙账户

账户之间权限平等。你可以看到本机账户列表，但只能管理当前账户，内容仍由各自密码和恢复码保护。

核心功能不收费

安全中心、恢复码、Touch ID、本地备份、保险柜和大文件能力都是免费功能，没有“高级版”门槛。

离线、安静、可控

适合证件照片、合同扫描件、项目备忘、家庭资料和不想进入网盘的个人档案。

安全事件看得见

登录、锁定、Touch ID、高级保护、导入导出和危险操作会写入本地加密安全日志，且不记录正文、文件名或恢复码。

不是云笔记的弱化版，而是本地隐私的完整版。

密笺把安全边界放在本机、密码、恢复码和备份上。它不需要服务器来证明自己安全，也不会把你的内容拿去换便利。

纯本地存储

笔记、账号、附件索引和保险柜文件都保存在本机。没有云同步，没有远程账号，也没有隐藏的分析脚本。

内容加密后落盘

应用写入磁盘的是加密数据。即使直接打开数据文件，也不会看到明文笔记或文件内容。

高级保护更克制

开启后自动锁定收紧到 1 分钟，并阻止复制、普通导出、共享导入导出、保险柜预览、保险柜导出和敏感文件名复制。

虚假密码

可为高级保护账户设置虚假密码：输入后进入临时虚假空间，或按你的设置直接销毁本地保险库数据。

纯免费，无付费开关

已经移除所有付费、授权、会员和购买逻辑。核心功能不再被“高级版”这种东西切开。

保险柜不是摆设。大照片库也能认真装。

新文件使用分片加密格式。导入大文件时不把整个文件塞进内存，导出也按流式解密写回，10GB 图片或素材包不会把界面拖死。

系统授权

通过 macOS 文件面板选择文件，按系统授予的安全访问权限读取和导出。

分片加密

每 4MB 作为一个加密块处理，避免一次性占用巨大内存。

后台导入

超过 64MB 的文件进入后台任务，主窗口继续响应搜索、编辑和切换。

流式导出

导出时逐块解密写入目标位置，兼容旧格式附件，也照顾新版本大文件。

账户平等，内容各自保护。

密笺适合一台 Mac 上的个人、多身份或家庭场景。账户之间彼此可见但不能互相删除或读取内容。

用户体验

  • 首次进入显示产品简介，应用内保留更新日志、法律说明和隐私声明。
  • 笔记支持置顶、收藏、归档、标签、Markdown 预览和加密分享文件。
  • 安全中心提供本地安全日志，记录登录、锁定、保护开关、导入导出和危险操作。
  • 保险柜支持预览、筛选、复制文件名、导出和删除。
  • Touch ID 可作为快捷解锁入口，闲置、休眠或切换用户后自动锁定。

管理能力

  • 新账户可直接创建，不再需要额外批准。
  • 只能删除当前账户；清空全部数据需要当前账户密码和确认文字。
  • 本地备份和还原不依赖云端，适合离线保存和迁移。
  • 恢复码只显示一次，其他账户不会变成读取用户内容的万能钥匙。

搜索更干净

历史输出和构建产物不再塞进项目索引。真正要被搜索的，是源码、网站和文档。

权限更克制

导入、导出、备份、还原都围绕系统文件授权做，不偷偷扩大文件访问范围。

测试守住核心

加密、恢复码、多账号、账户权限、保险柜和大文件路径都有自动测试覆盖。

把私密资料留给自己，不留给平台。

密笺适合保存个人笔记、证件照片、离线资料、项目备忘和不想进云端的文件。需要 macOS 14 或更高版本。
