为本地隐私而做

密笺不把你的内容交给服务器。它把安全边界放回这台 Mac、你的密码、你的恢复码和你的备份习惯上。

1

没有云端账户

应用不提供云同步、不上传笔记、不做行为分析，也没有广告和会员入口。

2

管理员看不到笔记

管理员负责批准注册、管理用户和备份还原，但不能直接查看普通用户的加密内容。

3

忘记密码也有边界

恢复码只显示一次。没有用户密码和恢复码时，管理员密码也不能变成万能钥匙。

保险柜可以认真装大文件

照片、PDF、压缩包、视频素材都可以进入独立保险柜。新版本使用分片加密，超大文件后台导入，导出时流式解密。

选择文件

从系统文件面板选择照片或文件，密笺显式使用系统授予的访问权限。

分片加密

按 4MB 分块处理，不再把 10GB 文件一次性读进内存。

后台导入

超过 64MB 的文件会放到后台加密，主界面仍可继续响应。

原件移除

成功进入保险柜后，应用会尝试删除原位置文件，并提示未能删除的项目。

账号、安全中心与恢复能力

密笺适合一台 Mac 上的个人或多人本地使用。每个用户有独立密码、恢复码和 Touch ID 快捷解锁状态。

日常使用

  • 笔记支持置顶、收藏、归档、标签和 Markdown 预览。
  • 保险柜文件可以筛选、预览、导出、复制文件名或删除。
  • 支持导出 Markdown、TXT 和加密共享文件。
  • 首次进入有简介，应用内保留更新日志和法律声明。

安全控制

  • 安全中心集中显示账号角色、Touch ID、高级保护和自动锁定。
  • 休眠、切换用户或闲置超时后自动锁定并清空内存内容。
  • 支持本地备份和从备份还原，不依赖云端恢复。
  • 主窗口禁止系统窗口共享，搜索只在解锁后的内存中进行。
