About

Local-first encrypted notes and file vault.

本地优先的加密记事本与文件保险柜。

  • Data stays on this Mac.数据留在这台 Mac 上。
  • Notes and vault files are encrypted before disk write.记事与保险柜文件会在落盘前加密。
  • No administrator account, no paid feature gate.没有管理员账号，也没有付费功能门槛。
README.md

Why CipherNotes为什么选择 CipherNotes

CipherNotes is designed for private notes, IDs, contracts, scans, family files, project archives, and anything you do not want to place into a cloud notebook or drive. CipherNotes 适合私密记事、证件、合同、扫描件、家庭文件、项目归档，以及任何你不想放进云笔记或云盘的内容。

Clear privacy boundary清晰的隐私边界

No remote account, no analytics, no sync server, no paid switch. Your vault file and encrypted chunks stay local.没有远程账号、没有分析、没有同步服务器、没有付费开关。保险库文件和加密分片只保存在本地。

Notes and files together笔记与文件统一管理

Keep text, photos, PDFs, archives, scans, and large private files in one encrypted local workspace.把文字、照片、PDF、压缩包、扫描件和大文件都放在一个本地加密空间里。

Large-file path大文件路径

Vault files use 4 MB encrypted chunks. Large imports run in the background and exports stream back to disk.保险柜文件使用 4 MB 加密分片。大文件导入在后台运行，导出时按流写回磁盘。

Equal local accounts平等本地账户

Accounts are visible on this Mac, but each account can only manage itself. There is no master account.本机上可见多个账户，但每个账户只能管理自己，没有主账号。

Advanced protection高级保护

Hide previews, block copy/export/share paths, tighten auto-lock, and optionally configure a decoy password.隐藏预览、阻止复制/导出/分享、收紧自动锁定，还可以配置虚假密码。

Local security log本地安全日志

Security events are encrypted with the account and avoid note bodies, filenames, passwords, and recovery codes.安全事件与账户一起加密保存，不记录正文、文件名、密码或恢复码。

Privacy boundary

Stored locally本地存储

  • Encrypted note payloads加密的笔记内容
  • Encrypted vault file chunks加密的保险柜文件分片
  • Account display names账户显示名
  • Encrypted local security logs加密的本地安全日志
  • Optional local backups you create你自己创建的本地备份

Never uploaded by the app应用不会上传

  • Note titles or bodies笔记标题或正文
  • Photos, PDFs, archives, scans照片、PDF、压缩包、扫描件
  • Passwords or recovery codes密码或恢复码
  • Usage analytics or ads data使用分析或广告数据
  • Security log sensitive object names安全日志中的敏感对象名
