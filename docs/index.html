README.md

Why CipherNotes

CipherNotes is designed for private notes, IDs, contracts, scans, family files, project archives, and anything you do not want to place into a cloud notebook or drive.

Clear privacy boundary

No remote account, no analytics, no sync server, no paid switch. Your vault file and encrypted chunks stay local.

Notes and files together

Keep text, photos, PDFs, archives, scans, and large private files in one encrypted local workspace.

Large-file path

Vault files use 4 MB encrypted chunks. Large imports run in the background and exports stream back to disk.

Equal local accounts

Accounts are visible on this Mac, but each account can only manage itself. There is no master account.

Advanced protection

Hide previews, block copy/export/share paths, tighten auto-lock, and optionally configure a decoy password.

Local security log

Security events are encrypted with the account and avoid note bodies, filenames, passwords, and recovery codes.

Privacy boundary

Stored locally

  • Encrypted note payloads
  • Encrypted vault file chunks
  • Account display names
  • Encrypted local security logs
  • Optional local backups you create

Never uploaded by the app

  • Note titles or bodies
  • Photos, PDFs, archives, scans
  • Passwords or recovery codes
  • Usage analytics or ads data
  • Security log sensitive object names
