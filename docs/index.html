不是云笔记的弱化版，而是本地隐私的完整版。

密笺把安全边界放在本机、密码、恢复码和备份上。它不需要服务器来证明自己安全，也不会把你的内容拿去换便利。

本

纯本地存储

笔记、账号、附件索引和保险柜文件都保存在本机。没有云同步，没有远程账号，也没有隐藏的分析脚本。

密

内容加密后落盘

应用写入磁盘的是加密数据。即使直接打开数据文件，也不会看到明文笔记或文件内容。

免

纯免费，无付费开关

已经移除所有付费、授权、会员和购买逻辑。核心功能不再被“高级版”这种东西切开。

保险柜不是摆设。大照片库也能认真装。

新文件使用分片加密格式。导入大文件时不把整个文件塞进内存，导出也按流式解密写回，10GB 图片或素材包不会把界面拖死。

系统授权

通过 macOS 文件面板选择文件，按系统授予的安全访问权限读取和导出。

分片加密

每 4MB 作为一个加密块处理，避免一次性占用巨大内存。

后台导入

超过 64MB 的文件进入后台任务，主窗口继续响应搜索、编辑和切换。

流式导出

导出时逐块解密写入目标位置，兼容旧格式附件，也照顾新版本大文件。

管理员管秩序，用户管内容。

密笺适合一台 Mac 上的个人、多身份或家庭场景。管理员能维护账号和数据生命周期，但普通用户的内容仍然由自己的密码和恢复码保护。

用户体验

  • 首次进入显示产品简介，应用内保留更新日志、法律说明和隐私声明。
  • 笔记支持置顶、收藏、归档、标签、Markdown 预览和加密分享文件。
  • 保险柜支持预览、筛选、复制文件名、导出和删除。
  • Touch ID 可作为快捷解锁入口，闲置、休眠或切换用户后自动锁定。

管理能力

  • 管理员密码可以修改，管理员和普通用户权限边界更完整。
  • 支持删除任意用户及其数据，也支持清空全部数据重新开始。
  • 本地备份和还原不依赖云端，适合离线保存和迁移。
  • 恢复码只显示一次，管理员密码不会变成读取用户内容的万能钥匙。
搜

搜索更干净

历史输出和构建产物不再塞进项目索引。真正要被搜索的，是源码、网站和文档。

权

权限更克制

导入、导出、备份、还原都围绕系统文件授权做，不偷偷扩大文件访问范围。

稳

测试守住核心

加密、恢复码、多账号、管理员能力、保险柜和大文件路径都有自动测试覆盖。

把私密资料留给自己，不留给平台。
