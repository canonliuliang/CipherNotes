# 密笺（CipherNotes）

纯本地的 macOS 加密记事本与照片/文件保险柜。支持管理员密码、本地用户注册和用户登录，没有云端账户、广告、分析或网络请求。

## 隐私设计

- 笔记标题和正文使用 AES-256-GCM 加密后写入磁盘；照片、PDF、压缩包等保险柜文件会以独立加密文件块保存，不依附于某条笔记。
- 第一次启动只创建管理员密码；管理员密码用于批准注册新用户和升级旧保险库，不能直接查看用户笔记。
- 每个用户用自己的“账户 + 用户密码”登录；为便于登录页选择账户，账户显示名会保存在本机保险库文件中，同时保留带独立随机盐的用户名摘要用于校验。同名账户仍会阻止创建，避免登录和恢复时混淆。
- 用户密码经 PBKDF2-HMAC-SHA256（310,000 轮及独立随机盐）派生，仅用于包裹该用户自己的随机保险库密钥；用户密码本身从不保存。应用不限制用户名格式、用户名长度或密码长度，空用户名会显示为“未命名账户”，空密码也允许。
- 注册和旧数据迁移时可选择启用 Touch ID；登录页选择账户后，可用密码或 Touch ID 登录。每个账户的 Touch ID 快捷解锁密钥分别保存在 macOS 钥匙串中，但 Touch ID 本身是这台 Mac 的系统级验证，不区分密笺用户；多人共用设备时建议关闭。
- 注册、迁移或手动更新时会生成用户恢复码；恢复码只显示一次，应用不会保存明文恢复码。忘记用户密码时，可以用恢复码重设密码并保留笔记。
- 支持把照片或任意文件移入独立“保险柜”。移入时会先加密保存，成功后删除原照片/原文件；保险柜里的图片可预览，文件可导出或删除。也支持导出和导入 `.ciphernote` 加密共享笔记；共享文件使用发送者输入的共享密码加密，应用不会保存共享密码。
- 管理员可删除任意用户，但不能查看该用户笔记；删除会移除该用户的加密数据和 Touch ID 快捷解锁项，数据不可恢复。
- 登录后可从“安全中心”集中查看当前账号角色、Touch ID、高级数据保护、自动锁定、恢复码、备份还原和本地数据位置。
- 保险柜文件使用分片加密存储；超大图片/文件会在后台按块加密和流式导出，不需要一次性读入内存。图片预览会限制超大图片并缓存缩略预览；导入、导出、备份、还原会显式使用系统文件面板授予的访问权限，减少权限报错和界面卡顿。
- 休眠、切换用户或达到闲置时限后自动清除内存中的笔记并锁定。
- 主窗口设置为禁止系统窗口共享。搜索只在解锁后的内存中进行。
- 密笺是纯免费本地应用，没有会员、广告、购买入口或付费门槛；高级数据保护、恢复码重生成、保险柜、导入导出等功能都可直接使用。

> 忘记某个用户的登录密码且没有保存恢复码时，该用户的笔记无法恢复。管理员密码不是万能解锁钥匙，这是本地零知识设计的必要结果。

## 运行

需要 macOS 14 或更高版本和 Xcode 15 或更高版本。

```sh
swift run --scratch-path /tmp/ciphernotes-run CipherNotes
```

也可以用 Xcode 打开 `Package.swift`，选择 CipherNotes scheme 后运行。

从旧版升级时，应用会显示“升级旧保险库”：创建一个新的管理员密码，再输入旧版用户名和旧版主密码。旧密码会成为该用户的登录密码，原有笔记不会丢失。如果忘记旧密码，也可以选择“跳过，清空旧数据并重新开始”。

## 测试

```sh
swift test --scratch-path /tmp/ciphernotes-test
```

## 打包

```sh
Packaging/build-release.sh
```

脚本会自动运行测试、构建 release、更新 `outputs/密笺安装器.pkg`、`outputs/密笺-macOS.zip`、`outputs/使用说明.md`、`outputs/产品介绍.html` 和 `outputs/密笺图标.png`。为减少 Spotlight 搜索噪音，`outputs` 不再保留展开后的 `.app` 包；需要 App 时解压 zip 即可。

保险库默认保存在 `~/Library/Application Support/CipherNotes/vault.json`，保险柜文件的加密块保存在同目录的 `Attachments/` 中。主文件中只有账户显示名、加密负载、随机盐、管理员验证信息、恢复码包裹后的密钥和用户密码包裹后的密钥，没有明文标题、正文、保险柜文件名、密码或恢复码。保险柜文件块中也只有密文。共享文件中的标题、正文和发送者用户名同样会被加密。
